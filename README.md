# YCP-ChimeBot

GASを使ったSlackチャイムBot

## 簡単な説明

Slackに指定した文章を指定した時間に送信する



## 機能

- トリガーの設定・消去
- 実行日が平日か判定(平日のみ送信したい為)
- 指定したメッセージを指定時間にJSON送信
